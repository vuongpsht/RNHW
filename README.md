### Quick installation

run below command to set up yarn & android with jetifier & ios with pod & lint & run
```shell
yarn setup
```

### Test deeplink

1. Country

usage
    ```shell
    yarn dlcountryIOS + code
    ```
    example
    ```shell
    yarn dlcountryIOS VN
    ```

2. Continent

usage
